URL Shortener

This URL shortener takes a valid URL as a parameter and returns a JSON object with the original URL and the shortened version. The user can then use the shortened URL to access the original page.

Enter a URL like one of these into your browser URL bar:

https://matty22urlshortener.herokuapp.com/new/http://www.google.com

You should receive a response that looks similar to this:

{"original_url": "https://www.google.com", "short_url": "https://matty22urlshortener.herokuapp.com/52"}
